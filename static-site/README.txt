
# Insight Smart Hotel - Versión Estática HTML/CSS

## Descripción
Esta es una versión puramente HTML/CSS del sitio web de Insight Smart Hotel, diseñada para ser ejecutada directamente desde una carpeta sin necesidad de servidores o instalaciones adicionales.

## Instrucciones de Uso
1. Simplemente abra el archivo index.html en cualquier navegador web moderno.
2. No se requiere ninguna instalación o configuración adicional.

## Contenido
- index.html: Archivo principal del sitio web
- styles.css: Todos los estilos del sitio
- /lovable-uploads/: Carpeta que contiene las imágenes del sitio

## Notas
- Esta versión es completamente estática y no depende de JavaScript complejo o React.
- Incluye una mínima cantidad de JavaScript para funcionalidades básicas como el menú móvil y el desplazamiento suave.
- Todas las secciones del sitio son responsive y se adaptan a diferentes tamaños de pantalla.
